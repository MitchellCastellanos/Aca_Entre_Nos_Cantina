# 🍻 Acá Entre Nos – Sitio Web

Sitio web de **Acá Entre Nos – Cantina regional mexicana**  
Desarrollado por **GABAN Solutions**

## 📁 Estructura del proyecto

- `index.html`
- `404.html`
- `robots.txt`
- `sitemap.xml`
- `site.webmanifest`
- `browserconfig.xml`
- `css/style.css`
- `js/main.js`
- `js/events.js`
- `js/reservations.js`

## ✏️ Cómo editar el sitio

### Eventos
Editar en:
`js/events.js`

### Promos del carrusel
Editar en:
`index.html`

### Galería
Reemplazar imágenes en:
`images/gallery/`

### Ubicación
Editar sección `#ubicacion` en:
`index.html`

### Instagram
Buscar:
`acaentrenosmexico`

### Estilos
Editar en:
`css/style.css`

## ⚙️ Reservaciones
Se conectan con Google Sheets usando:
`js/reservations.js`
